about this page

this page is generated by watching the bluesky jetstream for posts with bandcamp links. it updates every few minutes, feel free to refresh for updated content. at some point i'll make it so you can opt-in to new content without having to refresh.

if you want to link to the page with filters already filled out, you can use e.g. ?tags=metal. also, the tag filter input should autocomplete, but it doesn't on Firefox Mobile unfortunately. it's a six year old bug that maybe will get fixed someday.

i would like to do more with being able to tell what's playing, maybe do some auto-playing of the next thing in the list, but i'm currently limited by the using the default bandcamp embeds and cross-origin protection, so once that iframe loads i got no clue what's going on in it.

feel free to @ me if you have any requests.
